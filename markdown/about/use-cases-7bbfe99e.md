# Use cases

Created: 2026-05-31T01:45:49Z

## Note

Mianotes is useful whenever humans and agents need a shared, searchable, local knowledge base.

## Developers using AI agents

Use Mianotes as the documentation layer for Codex, Claude Code, Cursor, OpenClaw, VS Code agents, Slack bots, and local automation.

Every local agent has a goal, and every agent needs a place to report back. Mianotes gives agents a local workspace where they can explain what they did, how they did it, what changed, and what the outcome was.

Agents can save:

* Task summaries
* Implementation notes
* Decisions and trade-offs
* Changed files and why they changed
* Debugging findings
* Test results
* Questions and proposals
* Project context for the next agent run

This gives the person orchestrating the work a clear record of what happened: what their agents are doing, how they are doing it, and the result of each task. All of that context is captured and centralised in one place.

## AI agents and tools

Agents can use Mianotes as long-term memory that stays under your control.

An agent can:

* Create a folder for a task
* Add notes as it works
* Convert source files to Markdown
* Tag notes for discovery
* Search earlier notes before making changes
* Update a note when new evidence arrives
* Leave comments for review or handoff

## Students

Students can use Mianotes to turn study material into clean, searchable notes. Upload lecture slides, book pages, whiteboards, images, handwritten notes, or audio recordings, then ask Mia to structure the material as Markdown.

* Biology revision
* History essays
* Maths examples
* Lecture notes

Useful Mia prompts include:

```text
Mia turn this into revision notes with definitions and examples
Mia extract the key dates and people
Mia make flashcard-style questions from this note
```

## Small teams

Small teams can use Mianotes as a shared local knowledge base.

Typical folders:

* Spec and PRD files
* Research
* Meeting notes
* Internal decisions
* Roadmaps
* Launch plans

Team members can add tags across folders, comment on notes, and create read-only share links when someone outside the workspace needs access to a single note.

## Researchers

Researchers can use Mianotes to keep links, papers, documents, notes, prompts, and summaries in one searchable place.

Mianotes is especially useful when source material is messy because URL and file ingestion can preserve the original source next to the generated Markdown note.

## Families and everyday use

Mianotes also works as a private home knowledge base.

Examples:

* Homework
* School documents
* Trip plans
* Medical appointment notes
* Images of forms or letters

The value is not only AI. The value is that the final note is readable text, stored locally, and easy to search.
